Tugas Akhir Praktikum Pemrograman Web  







Tugas akhir ini membuat kalkulator berupa web yang responsive untuk mobile dan desktop.  

Kalkulator ini sama seperti kalkulator pada umumnya yaitu mengikuti perhitungan matematika dan akan menampilkan 5 history perhitungan terakhir.  

Kalkulator ini juga memiliki memory function yang akan memudahkan dalam menghitung jika terdapat perhitungan yang panjang, sehingga dapat menyimpan dan memanggil angka tanpa harus mengetik ulang.
